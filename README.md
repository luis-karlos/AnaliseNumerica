# Análise Numérica

Este repositório contém os códigos desenvolvidos na disciplina de **Análise Numérica** do Programa de Pós-Graduação em Matemática da Universidade Federal de Juiz de Fora (UFJF).

Os notebooks e scripts aqui presentes foram elaborados como parte das atividades propostas pelo professor **Sandro Rodrigues Mazorche**.

## Sobre a disciplina

A disciplina aborda métodos numéricos para resolução de problemas matemáticos.

## Estrutura do repositório

O repositório está organizado em pastas, cada uma correspondente a uma atividade da disciplina. A estrutura geral é descrita abaixo:

- `Atividades`: Cada pasta de atividade contém:
  - `Códigos`: Implementados em Jupyter Notebook (`.ipynb`), abordando os métodos numéricos propostos.
  - `PDF da atividade`: Documento com a descrição da atividade proposta.
  - `PDF da resolução`: Documento com a resolução da atividade.

## Instalação das Dependências

Para instalar as bibliotecas Python necessárias, certifique-se de que o ambiente virtual está ativado e execute o comando abaixo:

```bash
pip install -r requirements.txt
```

Isso garantirá que todas as dependências listadas no arquivo `requirements.txt` sejam instaladas corretamente.
